# Política de Privacidad

La extensión **Monitor de Foros Google** no recopila, almacena ni comparte datos personales de los usuarios.

## Uso de la información

- La extensión únicamente consulta contenido público disponible en foros de soporte de Google.
- No accede a cuentas de usuario.
- No recopila información personal.
- No realiza seguimiento del comportamiento del usuario.

## Almacenamiento de datos

- La configuración del usuario se guarda únicamente de forma local en el navegador mediante `chrome.storage`.
- No se envían datos a servidores externos.

## Compartición de datos

- No se comparten datos con terceros.
- No se venden datos.

## Seguridad

La extensión no ejecuta código remoto y todo su funcionamiento está contenido dentro del paquete instalado en el navegador.
